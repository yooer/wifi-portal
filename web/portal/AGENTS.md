# tools/ikuai-portal/web/portal/ Agent 指南

本目录维护连网访客进行身份校验的 Portal 验证页。

## 职责

- 呈现完美的苹果自适应轻量 UI。
- 处理访客接收并提交验证码，确保爱快、Mikrotik 等路由器的上线参数可以无损透传并完成算签。

## 约束

- **极致响应性**：不得引入大体积前端依赖或重型类库，确保移动端弱网下秒级开屏。
- **自适应系统时间**：明暗模式应兼顾系统 `prefers-color-scheme` 与当地当前时间（夜晚自动变暗，白天自动白润），优化访客在深夜灯光下的视觉体验。
- **输入法体验**：必须在验证码框加入 `inputmode="numeric"` 强制调用数字大键盘，按钮和文本需支持手势高亮。
