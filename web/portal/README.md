# tools/ikuai-portal/web/portal

本目录是访客通过手机或笔记本电脑连接 WiFi 时，所弹出并浏览的 Captive Portal 认证界面。

## 核心设计

- **苹果极简风**：高斯模糊微发光拟态，精心调整的卡片和按钮大小。
- **自动明暗模式**：白天（06:00 - 18:00）使用温润浅白主题，夜间（18:00 - 次日06:00）或系统开启 Dark 模式时自适应转换极夜深色主题。
- **无感数字键盘**：验证码输入框完美支持 `inputmode="numeric"` 及 `pattern="[0-9]*"`，点击秒弹纯大数字键盘，拒绝繁琐英文干扰。
- **防局部放大 (Zoom-Proof)**：表单字体严格保持为 `16px` 以上，解决 iOS 端点击输入自动拉近屏幕的闪烁缺陷。

## 当前文件

- `index.html` - 苹果风 Portal HTML5 认证页结构。
- `style.css` - 自适应明暗、高斯模糊美学样式及 iOS 质感核心。
- `portal.js` - 请求验证码冷却限频、爱快与多品牌算签上线重定向控制逻辑。
