# tools/ikuai-portal/web/admin

本目录是超级管理员与商户共用的后台管理控制面板前端代码。

## 设计风格

- **主题色**：极简白加翡翠绿 (Teal Green, `#00bfa5`)。
- **布局**：顶部悬浮水平导航菜单，摒弃传统侧栏。
- **体验**：圆角微阴影的卡片、iOS 开关、细线高光、丝滑过渡动画。

## 当前文件

- `index.html` - 后台管理大厅与模态弹窗页面。
- `style.css` - 翡翠绿配色风格与响应式排版样式。
- `admin.js` - Tab 切换、CRUD 短信通道与财务计费业务数据渲染逻辑。
