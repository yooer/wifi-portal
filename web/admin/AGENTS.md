# tools/ikuai-portal/web/admin/ Agent 指南

本目录维护超级管理员与商户管理后台的 UI 界面的样式与动态数据交互。

## 职责

- 保证管理后台的极简高拟真视觉设计的一致性。
- 通过 Ajax 请求 `/api/admin/*` 接口更新并渲染最新的短信池通道、充值记账和连网数据。

## 约束

- **视觉一致性**：必须保持顶部导航条的激活指示横线、白底圆角卡片、柔和阴影，与商户操作区的极简苹果风相协调，避免引入与项目风格冲突的任何红蓝灰杂色。
- **敏感信息隐藏**：在管理表格或修改表单中，展示 Secret、Password 或 AppKey 必须用 `******` 遮蔽；如果用户未修改该字段，提交给后端的表单必须保留 `******` 由后端进行非覆盖合并，不得回写为星号明文。
