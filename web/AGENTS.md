# tools/ikuai-portal/web/ Agent 指南

本目录维护 WiFi 认证系统的静态前端网页资源。

## 职责

- 负责前端界面的视觉布局、微交互动作和静态文件的结构层级。

## 约束

- **嵌入性**：这些文件是由 Go 后端通过 `go:embed` 统一打包进可执行文件的。修改静态资源后，**必须重新运行 `go build` 编译出新二进制**以使改动生效。
- **无状态交互**：在编写前端时，不依赖复杂的脚手架框架，仅使用原生 HTML/CSS/JS 进行高阶视觉表现，保持代码的高响应性与低依赖。
- **商户隔离**：保证前台交互时的 API 调用带有正确的 session token，避免越权操作。
